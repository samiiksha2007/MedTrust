# MedTrust

MedTrust is a college mini project that demonstrates the integration of
Machine Learning, Blockchain, and Web Technologies to build a secure and
trustworthy healthcare support system.

## Project Structure
medtrust/ ├─ blog/        # Blog / landing website ├─ dashboard/   # React dashboard ├─ backend/     # FastAPI backend ├─ blockchain/  # Smart contracts

## Tech Stack

- Frontend: React (Vite), HTML, CSS, JavaScript
- Backend: FastAPI (Python)
- Database: MySQL
- Blockchain: Solidity (Ethereum test network)
- Machine Learning: External ML APIs

## Overview

1. Users visit the blog and click "Try it now"
2. Users are redirected to the dashboard
3. Medical data is submitted via forms
4. Backend forwards data to ML APIs
5. Prediction results are returned
6. Hash of the medical data is stored on blockchain
7. Actual data is stored in the database
